\chapter{Simulation}
\label{chap:simulation}

This chapter describes how scenarios are generated from the trained neural encoder--decoder model. The purpose is no longer only to reconstruct a curve at a fixed date, but to simulate future latent states and decode them into full zero-coupon bond curves. This provides the dynamic objects needed for the pricing analysis carried out later.

\section{Simulation of Latent Dynamics and Curve Reconstruction}
\label{sec:simulation-latent-dynamics}

We use the same swap data as in the previous chapters. Let \(S_0 \in \mathbb{R}^8\) denote an observed market swap curve at the initial date, corresponding to the eight quoted maturities used in the model. In the implementation, \(S_0\) is selected either as the first row in the dataset or as the row corresponding to a specified date. The trained encoder maps this curve to the latent initial state
\[
z_0 = E(S_0) \in \mathbb{R}^\ell,
\]
where \(\ell\) denotes the latent dimension.

Starting from \(z_0\), the latent state is simulated forward under the risk-neutral dynamics introduced in Chapter ????,
\[
dz_t = \mu(z_t)\,dt + \sigma(z_t)\,dW_t^Q,
\]
where \(W_t^Q\) is an \(\ell\)-dimensional Brownian motion under the risk-neutral measure \(Q\). In the implementation, the drift and short-rate functions are given by the trained networks
\[
\mu(z)=K(z), \qquad r(z)=R(z),
\]
while the network \(H\) returns state-dependent volatilities and correlations. These are used to construct the matrix \(L(z_t)\), which enters directly in the simulation of the latent process.

The dynamics are discretized using the Euler--Maruyama scheme. For a time step \(\Delta t > 0\) and grid points \(t_n = n\Delta t\), the simulated paths satisfy
\[
z_{t_{n+1}}
=
z_{t_n}
+
\mu(z_{t_n})\,\Delta t
+
L(z_{t_n})\,\Delta W_n,
\qquad
\Delta W_n \sim \mathcal{N}(0,\Delta t\,I_\ell).
\]

From the simulated short-rate path
\[
r_{t_0}, r_{t_1}, \dots, r_{t_N},
\]
the corresponding money-market discount process is constructed. In continuous time,
\[
D_t = \exp\left(-\int_0^t r_s\,ds\right).
\]
Since the short rate is only observed on the simulation grid, the integral is approximated numerically using the trapezoidal rule:
\[
\int_0^{t_n} r_s\,ds
\approx
\sum_{k=0}^{n-1} \frac{r_{t_k}+r_{t_{k+1}}}{2}\,\Delta t,
\]
which gives
\[
D_{t_n}
\approx
\exp\left(
-
\sum_{k=0}^{n-1} \frac{r_{t_k}+r_{t_{k+1}}}{2}\,\Delta t
\right).
\]

For each simulated latent state \(z_{t_n}\), the corresponding zero-coupon bond curve is obtained through the decoder described in Chapter~\ref{chap:framework}. More precisely, for each realized state \(z_{t_n}\), the functions \(A_{z_{t_n}}(\tau)\) and \(B_{z_{t_n}}(\tau)\) are obtained by solving the no-arbitrage ODE system numerically, after which bond prices are recovered from
\[
P(z_{t_n},\tau)
=
\exp\bigl(A_{z_{t_n}}(\tau)-B_{z_{t_n}}(\tau)G(z_{t_n},\tau)\bigr).
\]

Hence, each simulated path produces not only a short-rate path and pathwise discount factors, but also a full sequence of zero-coupon bond curves across maturities.


\section{Monte Carlo Pricing of Swaptions}
\label{sec:swaption-pricing}

Having generated simulated latent paths and the associated zero-coupon bond curves, we can now price interest-rate derivatives by Monte Carlo under the risk-neutral measure. In this section, we consider European swaptions.

Let \(T_e\) denote the option expiry, let \(n\) denote the number of fixed-leg payments of the underlying swap, and let \(\delta>0\) denote the accrual period. In the implementation, we use annual accrual, so \(\delta=1\). The payment dates of the underlying swap are therefore
\[
T_e+\delta,\; T_e+2\delta,\; \dots,\; T_e+n\delta.
\]
The corresponding swap tenor is \(n\delta\).

The initial strike is determined from the time-zero discount curve. Given the decoded bond prices \(P(0,T)\), the swap annuity at time \(0\) is
\[
A_0
=
\delta \sum_{j=1}^{n} P(0,T_e+j\delta),
\]
and the corresponding forward swap rate is
\[
F_0
=
\frac{P(0,T_e)-P(0,T_e+n\delta)}{A_0}.
\]
For an at-the-money swaption, the strike is chosen as
\[
K = F_0.
\]

For each simulated path \(m=1,\dots,M\), the decoder delivers a full zero-coupon bond curve at expiry. Since the model is expressed in terms of time-to-maturity, this curve is written as
\[
\tau \mapsto P^{(m)}(T_e,\tau).
\]
The payment dates of the underlying swap correspond to the maturities
\[
\delta,\; 2\delta,\; \dots,\; n\delta
\]
measured from \(T_e\). The pathwise annuity at expiry is therefore
\[
A^{(m)}(T_e)
=
\delta \sum_{j=1}^{n} P^{(m)}(T_e,j\delta),
\]
and the corresponding spot-starting swap rate is
\[
S^{(m)}(T_e)
=
\frac{1-P^{(m)}(T_e,n\delta)}{A^{(m)}(T_e)}.
\]

Let \(\mathcal N>0\) denote the notional. The payer swaption payoff on path \(m\) is
\[
X^{(m)}_{T_e}
=
\mathcal N\,A^{(m)}(T_e)\max\bigl(S^{(m)}(T_e)-K,0\bigr),
\]
while the receiver swaption payoff is
\[
X^{(m)}_{T_e}
=
\mathcal N\,A^{(m)}(T_e)\max\bigl(K-S^{(m)}(T_e),0\bigr).
\]

The payoff is discounted back to time \(0\) using the pathwise money-market discount factor from Section~\ref{sec:simulation-latent-dynamics}. Denoting this factor on path \(m\) by
\[
D^{(m)}_{T_e}
=
\exp\left(-\int_0^{T_e} r_s^{(m)}\,ds\right),
\]
the discounted present value on path \(m\) is
\[
V_0^{(m)} = D^{(m)}_{T_e} X^{(m)}_{T_e}.
\]
The time-zero swaption price is then estimated by the Monte Carlo average
\[
\widehat V_0
=
\frac{1}{M}\sum_{m=1}^{M} V_0^{(m)}.
\]

To assess numerical uncertainty, we also report the sample standard deviation
\[
\widehat \sigma_{\mathrm{MC}}
=
\sqrt{\frac{1}{M-1}\sum_{m=1}^{M}\left(V_0^{(m)}-\widehat V_0\right)^2}
\]
and the associated Monte Carlo standard error
\[
\widehat{\mathrm{SE}}_{\mathrm{MC}}
=
\frac{\widehat \sigma_{\mathrm{MC}}}{\sqrt{M}}.
\]

Finally, the Monte Carlo price is converted into a model-implied normal volatility using the Bachelier swaption formula. For a payer swaption, the Bachelier price is
\[
V_0^{\mathrm{Bach}}
=
\mathcal N A_0
\left[
(F_0-K)\Phi(d) + \sigma_N \sqrt{T_e}\,\phi(d)
\right],
\qquad
d=\frac{F_0-K}{\sigma_N\sqrt{T_e}},
\]
where \(\Phi\) and \(\phi\) denote the standard normal distribution and density functions, respectively, and \(\sigma_N\) is the normal volatility.

For a receiver swaption, the corresponding formula is
\[
V_0^{\mathrm{Bach}}
=
\mathcal N A_0
\left[
(K-F_0)\Phi(-d) + \sigma_N \sqrt{T_e}\,\phi(d)
\right].
\]
The model-implied normal volatility is then obtained by numerically inverting the Bachelier formula so that the analytical price matches the Monte Carlo estimate \(\widehat V_0\).

In the implementation, the time-zero quantities \(F_0\) and \(A_0\) are computed directly from the decoded initial discount curve, while the pathwise quantities \(S^{(m)}(T_e)\) and \(A^{(m)}(T_e)\) are computed from the decoded discount curve at expiry on each simulated path. Because the pricing routine extracts bond prices directly from the model grids, the option expiry \(T_e\) must lie on the simulation time grid, while the payment dates \(j\delta\) must lie on the decoder maturity grid. In the numerical examples below, we consider in particular at-the-money payer swaptions, for which the strike is set equal to the time-zero forward swap rate.

Overall, the pricing pipeline can be summarized schematically as
\[
z \;\longrightarrow\; P \;\longrightarrow\; (F_0,A_0) \;\longrightarrow\; (S(T_e),A(T_e)) \;\longrightarrow\; X(T_e) \;\longrightarrow\; V_0 \;\longrightarrow\; \sigma_N.
\]

%========================================================
\chapter{Simulation Diagnostics: Baseline vs.\ Stable}
\label{chap:sim_comparison}
%========================================================

The previous chapter established that the baseline autoencoder satisfies the
theoretical conditions for existence and uniqueness of the latent SDE solution.
It also showed, in theory, why the baseline is \emph{not} guaranteed to produce
well-behaved simulated paths: the drift matrix $M$ can have positive eigenvalues
and the volatility network can output large values at states far from the training
cloud.
This chapter turns those theoretical warnings into a concrete empirical demonstration.

We simulate $N = 500$ paths over a 10-year horizon ($\Delta t = 1/12$, 120 monthly
steps) using both the baseline and stable checkpoints trained for 3\,500 epochs
($\ell = 2$, EUR swap curves).
\emph{Identical} Brownian increments are used for both models, so any difference
between the paths is attributable solely to the learned network parameters, not to
stochastic variation.
The figures and statistics below are generated by the script
\texttt{compare\_simulation\_variants.py}.


%========================================================
\section{Problems with the baseline model during simulation}
\label{sec:sim_baseline_problems}
%========================================================

Three distinct failure modes can arise when the baseline model is used for
forward simulation.  They are ordered by severity: the third (explosive drift)
is catastrophic and renders the model entirely unusable for pricing.

%--------------------------------------------------------
\subsection{Unstable drift: non-mean-reverting latent factors}
\label{subsec:unstable_drift}
%--------------------------------------------------------

The baseline drift network performs the affine map
\[
    \mathcal{K}(z) = Mz + N, \qquad M \in \mathbb{R}^{\ell \times \ell},
\]
where $M$ and $N$ are learned without any constraint on the eigenvalues of $M$.
As shown in Proposition~2 of Chapter~\ref{chap:stable_ae} (the Stable chapter),
mean reversion toward the fixed point $z^* = -M^{-1}N$ requires \emph{all}
eigenvalues of $M$ to have strictly negative real parts, i.e.\ $M$ must be
a \emph{Hurwitz} matrix.

When $M$ has at least one eigenvalue with non-negative real part, the
corresponding component of the deviation $u_t = z_t - z^*$ grows without bound
as $t \to \infty$.
In the Euler--Maruyama discretisation this means the latent state can drift
arbitrarily far from the training cloud within a finite simulation horizon.

\paragraph{What we observe in the ep5000 baseline and stable models.}
After 5\,000 training epochs, the baseline drift matrix exhibits \emph{both} positive eigenvalues:
$\lambda_1 \approx -0.40$ (stable) but $\lambda_2 \approx +1.64$ (unstable).
The positive eigenvalue dominates, causing latent paths to diverge exponentially.
By contrast, the stable variant constrains $M = -(V^\top V + \varepsilon I)$, ensuring
all eigenvalues are strictly negative ($\lambda_1 \approx -0.014$, $\lambda_2 \approx -2.91$).
The empirical statistics from the simulation confirm this stark difference:
\begin{center}
\begin{tabular}{lrr}
\toprule
& \textbf{Baseline (ep5000)} & \textbf{Stable (ep5000)} \\
\midrule
$z_1$ mean  & $-5.6 \times 10^{16}$ & $-0.20$ \\
$z_1$ std   & $4.2 \times 10^{18}$  & $0.64$  \\
$z_1$ min   & $-1.8 \times 10^{20}$ & $-2.68$ \\
$z_1$ max   & $1.8 \times 10^{20}$ & $3.38$ \\
\midrule
$z_2$ mean  & $-5.2 \times 10^{16}$ & $-0.06$ \\
$z_2$ std   & $3.9 \times 10^{18}$  & $0.10$  \\
$z_2$ min   & $-1.7 \times 10^{20}$ & $-0.56$ \\
$z_2$ max   & $1.6 \times 10^{20}$ & $0.40$ \\
\midrule
$r$ mean    & $3449\%$ & $231\%$  \\
$r$ std     & $11535\%$ & $561\%$  \\
\bottomrule
\end{tabular}
\end{center}
The baseline latent state reaches values on the order of $10^{20}$ — essentially diverging
to floating-point infinity — making the model completely unusable for pricing.
The stable model, by contrast, maintains bounded paths with $|z| \leq 3.4$ and
realistic short rates near $2\%$, demonstrating the critical importance of the
structural constraints.

%--------------------------------------------------------
\subsection{Unbounded and unstable short rates}
\label{subsec:unstable_short_rate}
%--------------------------------------------------------

The short-rate network $\mathcal{R}(z)$ maps the latent state to a scalar via
a two-layer feedforward network with the centered soft-step activation
$\psi(x)=\sigma(x)-\tfrac{1}{2}\in(-\tfrac{1}{2},\tfrac{1}{2})$:
\[
    \tilde{r}(z)=W_2\,\psi(W_1 z),\qquad
    W_1\in\mathbb{R}^{h\times\ell},\;W_2\in\mathbb{R}^{1\times h},
\]
with no bias terms. Because $\psi$ is component-wise bounded in
$(-\tfrac{1}{2},\tfrac{1}{2})$, the output satisfies
$|\tilde{r}(z)|\leq\tfrac{1}{2}\|W_2\|_1$ for \emph{every} $z$.
This prevents the short rate itself from reaching infinity at any individual
time step.
However, once the latent state has diverged to $\|z\| \sim 10^{20}$, the
short-rate network is evaluated deep in its saturation regime: every path is
stuck at either $r_{\min}$ or $r_{\max}$, removing all cross-sectional and
time-series variation.

In the baseline simulation with ep5000, the short rate saturates at the hard limits
with mean $3449\%$ and standard deviation $11535\%$ --- a pathological distribution
completely divorced from economic reality and destroying all model utility.
This makes any discount factor or derivative price computed from these paths
economically meaningless.

%--------------------------------------------------------
\subsection{Compound effect: simulation is unusable for pricing}
\label{subsec:compound_effect}
%--------------------------------------------------------

The two problems above interact multiplicatively.
Once the latent state escapes the training cloud, the decoder $\mathcal{G}$ is
evaluated at a wildly out-of-distribution input, producing bond prices and swap
rates that bear no relation to observed market data.
Crucially, this happens \emph{quickly}: Figure~\ref{fig:comp_latent_paths} shows
that even the individual path spaghetti plot is dominated by extreme values,
while the stable paths remain on a reasonable scale.

The conclusion is stark: a model that fits the training data perfectly but has an
unstable drift is \emph{entirely unsuitable} for simulation-based derivative pricing.
Reconstruction accuracy is a necessary but far from sufficient condition for a
pricing model.


%========================================================
\section{How the stable model solves each problem}
\label{sec:sim_stable_solutions}
%========================================================

The stable variant addresses each of the three failure modes through targeted
structural modifications, detailed in Chapter~\ref{chap:stable_ae}.
Here we summarise each fix and connect it to what we see in the figures.

%--------------------------------------------------------
\subsection{Fix 1: Guaranteed mean reversion via the Hurwitz constraint}
\label{subsec:fix_hurwitz}
%--------------------------------------------------------

The stable drift replaces the unconstrained matrix $M$ with
\[
    M = -\!\left(V^\top V + \varepsilon I_\ell\right), \qquad
    V \in \mathbb{R}^{\ell \times \ell}, \quad \varepsilon > 0.
\]
Since $V^\top V$ is positive semi-definite and $\varepsilon I$ is strictly positive
definite, the matrix $M$ is \emph{negative definite} for every choice of $V$.
All eigenvalues satisfy $\lambda_i \leq -\varepsilon < 0$, so the drift always
pulls $z_t$ back toward the equilibrium $z^* = -M^{-1}N$.

In the 10-year simulation, the stable latent factors revert to a neighbourhood
of $z^*$ and remain there throughout (Table above, right column;
Figures~\ref{fig:comp_fans} and~\ref{fig:comp_short_rate_fan}).
The paths spread naturally from the initial point and then stabilise,
exactly as a well-posed Ornstein--Uhlenbeck-type process should.

%--------------------------------------------------------
\subsection{Fix 2: Bounded volatilities via sigmoid link}
\label{subsec:fix_sigma}
%--------------------------------------------------------

The baseline volatility $\sigma_i(z) = \exp([\mathcal{H}(z)]_i)$ is bounded
theoretically by the network architecture, but the practical upper bound
$e^{C_{\mathrm{pre}}}$ can be large and is not controlled.
The stable network replaces the $\exp$-link with a sigmoid-bounded parameterization:
\[
    \log\sigma_i(z) = \log\sigma_{\min}
        + (\log\sigma_{\max} - \log\sigma_{\min})\cdot
          \operatorname{sigmoid}(h_i(z) + b_i),
\]
so that $\sigma_i(z) \in (\sigma_{\min}, \sigma_{\max})$ for \emph{every} $z$,
including states far outside the training cloud.
With default bounds $\sigma_{\min} = 10^{-4}$ and $\sigma_{\max} = 2.0$,
the diffusion can never produce a shock larger than $\sigma_{\max}\sqrt{\Delta t}$
per step, which prevents runaway path growth even if the drift were imperfect.

In combination with the Hurwitz constraint this means the process
$z_t$ is confined to a compact region in probability for any finite horizon $T$.

%--------------------------------------------------------
\subsection{Fix 3: Positive-definite correlation matrix for any $\ell$}
\label{subsec:fix_pd}
%--------------------------------------------------------

The baseline correlation parameterization $\rho_{ij} = \tanh(\cdot)$ guarantees
each pairwise correlation is in $(-1, 1)$, but for $\ell \geq 3$ a set of valid
pairwise values can still produce a non-positive-definite (NPD) full correlation
matrix.
An NPD correlation matrix has no Cholesky factorization, meaning the simulation
loop would crash or produce complex-valued shocks.

The stable network uses the \emph{hyperspherical Cholesky} parameterization
\citep{Pinheiro1996}: $\ell(\ell-1)/2$ free angles $\theta_k \in (0, \pi)$ produced
by $\theta_k(z) = \pi \cdot \operatorname{sigmoid}(g_k(z))$ are mapped to a
lower-triangular factor $L_R$ via the recurrence
\[
    L_R[i,j] = \cos\theta_{i,j+1}\prod_{k=1}^{j}\sin\theta_{i,k}, \quad
    L_R[i,i] = \prod_{k=1}^{i}\sin\theta_{i,k},
\]
and $R = L_R L_R^\top$ is positive definite by construction for all $z$.
This makes the diffusion matrix $\Sigma(z) = \operatorname{diag}(\sigma(z))\,L_R(z)$
always well-defined and non-singular, regardless of the latent state.

for the entire horizon.
The stable short rate, by contrast, starts near the initial observed level and
drifts within a roughly $\pm 2$\% band --- consistent with the EUR rate environment
of the training data.

Figures~\ref{fig:comp_fans} and~\ref{fig:comp_short_rate_fan} display the same
information in a summary form that is more informative for risk assessment.

For the \textbf{stable model} (red), the fan structure follows the
expected pattern of an Ornstein--Uhlenbeck process:
\begin{enumerate}
    \item \textit{Early diffusion phase} ($t \lesssim 1$\,yr): uncertainty grows
    as paths spread from the common starting point $z_0$.
    \item \textit{Mean-reversion phase} ($t \gtrsim 2$\,yr): the Hurwitz
    constraint pulls all paths toward $z^*$, and the bands stabilise or even
    narrow.
    The long-run distribution is the stationary distribution of the SDE,
    which is a Gaussian with covariance determined by the Lyapunov equation
    $M\Sigma_\infty + \Sigma_\infty M^\top + \Sigma(z^*)^2 = 0$.
\end{enumerate}

For the \textbf{baseline model} (blue), the fan structure is entirely different:
\begin{enumerate}
    \item The 5--95 band expands exponentially from the start, reflecting the
    positive eigenvalue(s) of $M$.
    \item The short-rate fan (Figure~\ref{fig:comp_short_rate_fan}, blue)
    is almost perfectly flat after the first time step: every path is saturated at
    $\pm r_{\mathrm{scale}}$ and the $50^{\mathrm{th}}$ percentile median
    oscillates near zero, reflecting an equal split between positively and
    negatively divergent paths.
\end{enumerate}

The qualitative contrast between the two sets of fans is exactly what the theory
predicts: stable means bounded uncertainty that grows and then stabilises;
baseline means unbounded uncertainty that grows without limit.

We first need to check if this overholds the condition for existence and uniqueness of an SDE.

\begin{theorem}
\label{thm:existence_uniqueness_sde}
 Let $\mu:\mathbb{R}^\ell\rightarrow\mathbb{R}^\ell$ and $\sigma:\mathbb{R}^\ell\rightarrow\mathbb{R}^{\ell\times\ell}$, be functions where
$$|\mu(z)|+|\sigma(z)|\leq C(1+|z|),\;\;\forall z\in\mathbb{R}^\ell$$

for some constant $C$. Furthermore if the Lipschitz condition holds

$$|\mu(z_1)-\mu(z_2)|+|\sigma(z_1)-\sigma(z_2)|\leq D|z_1-z_2|,\;\;\forall z_1, z_2,\in\mathbb{R}^\ell$$

for some constant $D$. Then the SDE
$$dz_t=\mu(z_t)dt+\sigma(z_t)dW_t$$
Has a unique solution, $z_t$, which is adapted to the filtration generated by $W$ and
$$E\left[\int_0^T||z_t||^2dt\right]<\infty$$       
\end{theorem}




So under this two conditions, the process is well-behaved, unique, and cannot explode. So we will now check if our Neural Networks actually obide this


\begin{proposition}
Let $\mathcal{K}: \mathbb{R}^\ell \to \mathbb{R}^\ell$ and 
$\mathcal{H}: \mathbb{R}^\ell \to \mathbb{R}^{\ell + \ell(\ell-1)/2}$ denote the drift 
and volatility networks of the baseline model. Then both satisfy the linear growth 
and Lipschitz conditions of Theorem~\ref{thm:existence_uniqueness_sde}.
\end{proposition}

\begin{proof}
\textbf{Drift $\mathcal{K}$.}
The drift network performs the affine transformation
\[
\mathcal{K}(z) = Mz + N, \quad M \in \mathbb{R}^{\ell \times \ell},\; N \in \mathbb{R}^\ell.
\]
\emph{Linear growth.} By the triangle inequality and submultiplicativity of the operator norm:
\[
\|\mathcal{K}(z)\| = \|Mz + N\| \leq \|M\|\|z\| + \|N\| \leq \max(\|M\|, \|N\|)(1 + \|z\|).
\]
So linear growth holds with $C_1 = \max(\|M\|, \|N\|)$.

\emph{Lipschitz.} For any $z, z' \in \mathbb{R}^\ell$:
\[
\|\mathcal{K}(z) - \mathcal{K}(z')\| = \|M(z - z')\| \leq \|M\|\|z - z'\|.
\]
So Lipschitz holds with $L_1 = \|M\|$.

\textbf{Diffusion $\mathcal{H}$.}
The volatility network has one hidden layer with no bias,
\[
\mathcal{H}(z) = W_2\,\psi(W_1 z), \quad 
W_1 \in \mathbb{R}^{h \times \ell},\; 
W_2 \in \mathbb{R}^{(\ell + \ell(\ell-1)/2) \times h},
\]
where $\psi(x) = \mathrm{sigmoid}(x) - 0.5$ is the CenteredSoftStep activation 
and $h$ denotes the hidden dimension. The actual volatilities are recovered as 
$\sigma_i(z) = \exp([\mathcal{H}(z)]_i)$.

\emph{Step 1: Bound the activation.} Since $\mathrm{sigmoid}(x) \in (0,1)$ for all 
$x \in \mathbb{R}$:
\[
|\psi(x)| = |\mathrm{sigmoid}(x) - 0.5| < 0.5 \quad \forall\, x \in \mathbb{R},
\]
and therefore $\|\psi(W_1 z)\| \leq \frac{\sqrt{h}}{2}$ for all $z \in \mathbb{R}^\ell$.

\emph{Step 2: Bound the network output.} By submultiplicativity:
\[
\|\mathcal{H}(z)\| = \|W_2\,\psi(W_1 z)\| \leq \|W_2\|\cdot\frac{\sqrt{h}}{2} 
=: C_{\mathrm{pre}} < \infty.
\]
This bound is independent of $z$, so each component satisfies 
$[\mathcal{H}(z)]_i \in [-C_{\mathrm{pre}},\, C_{\mathrm{pre}}]$ for all $z$.

\emph{Step 3: Linear growth of $\sigma_\theta$.} Since $\exp$ is monotone increasing:
\[
\|\sigma_\theta(z)\| = \bigl\|\exp(\mathcal{H}(z))\bigr\| 
\leq \sqrt{\ell}\,e^{C_{\mathrm{pre}}} 
\leq \sqrt{\ell}\,e^{C_{\mathrm{pre}}}(1 + \|z\|).
\]
So linear growth holds with $C_2 = \sqrt{\ell}\,e^{C_{\mathrm{pre}}}$.

\emph{Step 4: Lipschitz continuity of $\sigma_\theta$.} For any $z, z' \in \mathbb{R}^\ell$, 
apply the mean value theorem to $\exp$. Since 
$\exp'(x) = \exp(x) \leq e^{C_{\mathrm{pre}}}$ on $[-C_{\mathrm{pre}}, C_{\mathrm{pre}}]$:
\[
|\sigma_i(z) - \sigma_i(z')| \leq e^{C_{\mathrm{pre}}}|[\mathcal{H}(z)]_i - [\mathcal{H}(z')]_i|.
\]
Summing over components and applying submultiplicativity, together with the fact that 
$\psi'(x) = \mathrm{sigmoid}(x)(1 - \mathrm{sigmoid}(x)) \leq \frac{1}{4}$:
\[
\|\sigma_\theta(z) - \sigma_\theta(z')\| 
\leq e^{C_{\mathrm{pre}}}\|\mathcal{H}(z) - \mathcal{H}(z')\| 
\leq e^{C_{\mathrm{pre}}}\|W_2\|\cdot\frac{\|W_1\|}{4}\cdot\|z - z'\|.
\]
So Lipschitz holds with $L_2 = \frac{e^{C_{\mathrm{pre}}}\|W_1\|\|W_2\|}{4}$.

\textbf{Conclusion.} Both $\mathcal{K}$ and $\mathcal{H}$ satisfy the linear growth and 
Lipschitz conditions with explicit constants
\[
C_1 = \max(\|M\|, \|N\|), \quad L_1 = \|M\|, \quad
C_2 = \sqrt{\ell}\,e^{C_{\mathrm{pre}}}, \quad 
L_2 = \frac{e^{C_{\mathrm{pre}}}\|W_1\|\|W_2\|}{4}, \quad 
C_{\mathrm{pre}} = \frac{\sqrt{h}}{2}\|W_2\|.
\]
Therefore, by Theorem~\ref{thm:existence_uniqueness_sde}, the baseline model SDE has a 
unique, non-explosive solution. The key architectural property enabling this result 
is the bounded activation $\psi \in (-0.5, 0.5)$, which confines the pre-exponential 
outputs of $\mathcal{H}$ to a bounded interval independent of $z$, preventing the 
diffusion from growing with the state.
\end{proof}


\section{Problem 2: Mean reversion?}

Then the next problem is if the drift is mean reverting, because nothing in the baseline model is guaranteeing that. 



\begin{proposition}
The drift $\mathcal{K}(z) = Mz + N$ is mean-reverting toward the fixed point
$z^* = -M^{-1}N$ if and only if all eigenvalues of $M$ have strictly negative
real parts.
\end{proposition}

\begin{proof}
\textbf{Setup.} Assume $M$ is invertible so that $z^* = -M^{-1}N$ is 
well-defined. Define the deviation process:
\[
u_t = z_t - z^*
\]
Substituting into the ODE $\dot{z}_t = Mz_t + N$:
\[
\dot{u}_t = Mz_t + N = M(u_t + z^*) + N = Mu_t + 
\underbrace{Mz^* + N}_{=0} = Mu_t
\]
So the dynamics reduce to the homogeneous linear ODE:
\[
\dot{u}_t = Mu_t, \quad u_0 = z_0 - z^*
\]
By \citet[][p.~129]{Hirsch2012}, the unique solution is:
\[
u_t = e^{Mt}u_0
\]
where $e^{Mt}$ denotes the matrix exponential.

\textbf{($\Rightarrow$)} Suppose all eigenvalues satisfy 
$\mathrm{Re}(\lambda_i) < 0$. By the Jordan decomposition, write 
$M = PJP^{-1}$ where $J$ is the Jordan normal form, so that 
$e^{Mt} = Pe^{Jt}P^{-1}$. Each Jordan block $J_k$ corresponding to 
eigenvalue $\lambda_k$ contributes terms of the form $t^j e^{\lambda_k t}$ 
for $j = 0, 1, \ldots$ Since $\mathrm{Re}(\lambda_k) < 0$ for all $k$, 
polynomial growth is dominated by exponential decay:
\[
|t^j e^{\lambda_k t}| = t^j e^{\mathrm{Re}(\lambda_k)t} \to 0 
\quad \text{as } t \to \infty.
\]
Therefore $e^{Jt} \to 0$ and hence $e^{Mt} \to 0$, which gives:
\[
\|u_t\| = \|e^{Mt}u_0\| \leq \|P\|\|e^{Jt}\|\|P^{-1}\|\|u_0\| \to 0.
\]
So $z_t = z^* + u_t \to z^*$, i.e.\ $z^*$ is asymptotically stable. This 
is consistent with \citet[][p.~175]{Hirsch2012}, who establish that an 
equilibrium point is asymptotically stable if and only if all eigenvalues 
of the system matrix have strictly negative real parts. Since 
$\mathcal{K}$ is already linear, no linearization is needed and the 
result applies directly.

\textbf{($\Leftarrow$)} Suppose some eigenvalue satisfies 
$\mathrm{Re}(\lambda_i) \geq 0$, and let $v_i$ be the corresponding 
eigenvector. Choose $u_0 = v_i$. Then:
\[
u_t = e^{Mt}v_i = e^{\lambda_i t}v_i
\]
and therefore:
\[
\|u_t\| = |e^{\lambda_i t}|\|v_i\| = 
e^{\mathrm{Re}(\lambda_i)t}\|v_i\| \geq \|v_i\| > 0 
\quad \forall\, t \geq 0.
\]
So $u_t \not\to 0$, meaning $z_t \not\to z^*$.

\textbf{Conclusion.} The drift $\mathcal{K}(z) = Mz + N$ is mean-reverting 
toward $z^*$ if and only if all eigenvalues of $M$ have strictly negative 
real parts:
\[
\mathrm{Re}(\lambda_i) < 0 \quad \forall\, i = 1, \ldots, \ell.
\]
This is equivalent to $M$ being a \emph{Hurwitz matrix} 
\citep[][p.~175]{Hirsch2012}. Since the baseline model places no 
constraint on $M$ during training, mean reversion is not guaranteed --- 
whether it holds depends entirely on the learned parameters.
\end{proof}

\subsection*{Enforcing the Hurwitz Condition}

While the Hurwitz condition $\mathrm{Re}(\lambda_i) < 0$ for all $i$ is 
necessary and sufficient for mean reversion of the drift, it is not 
straightforward to enforce directly during training, as it would require 
monitoring the eigenvalues of $M$ at every gradient step. Instead, we 
construct $M$ such that it is Hurwitz \emph{by construction}.

\begin{proposition}
Let $V \in \mathbb{R}^{\ell \times \ell}$ be any matrix and $\epsilon > 0$. 
Then the matrix
\[
M = -(V^\top V + \epsilon I)
\]
is Hurwitz.
\end{proposition}

\begin{proof}
We show that all eigenvalues of $M$ have strictly negative real parts.
Let $\lambda$ be any eigenvalue of $M$ with corresponding eigenvector 
$x \neq 0$. Then:
\[
Mx = \lambda x \implies -(V^\top V + \epsilon I)x = \lambda x
\]
Taking the inner product with $x$:
\[
-x^\top(V^\top V + \epsilon I)x = \lambda \|x\|^2
\]
Since $V^\top V$ is positive semi-definite (as $x^\top V^\top V x = 
\|Vx\|^2 \geq 0$ for all $x$) and $\epsilon I$ is strictly positive 
definite, we have:
\[
x^\top(V^\top V + \epsilon I)x = \|Vx\|^2 + \epsilon\|x\|^2 \geq 
\epsilon\|x\|^2 > 0
\]
Therefore:
\[
\lambda = -\frac{x^\top(V^\top V + \epsilon I)x}{\|x\|^2} \leq 
-\epsilon < 0
\]
Since this holds for any eigenvector $x$, all eigenvalues of $M$ satisfy 
$\lambda \leq -\epsilon < 0$, and in particular 
$\mathrm{Re}(\lambda) \leq -\epsilon < 0$. Hence $M$ is Hurwitz. 
\end{proof}

\begin{remark}
Note that $M = -(V^\top V + \epsilon I)$ is always symmetric and negative 
definite. This is a sufficient but not necessary condition for being 
Hurwitz --- a general Hurwitz matrix need not be symmetric. The 
parameterization is therefore slightly more restrictive than the general 
Hurwitz condition, but has the practical advantage that $M$ is guaranteed 
to be Hurwitz for any choice of $V$, allowing $V$ to be freely learned 
during training without any additional constraints. The parameter 
$\epsilon > 0$ ensures strict negative definiteness even when $V$ is 
singular, and controls the minimum rate of mean reversion.
\end{remark}

\section{Problem 3: Bounded and positive-definite diffusion}
\label{sec:stable_diffusion}
%========================================================

Even when the drift is mean-reverting, the Euler--Maruyama simulation can still produce implausible paths if the diffusion coefficient grows too large or if the correlation matrix ceases to be positive definite at states far from the training cloud.
The baseline diffusion network $\mathcal{H}$ exhibits two potential weaknesses that become practically relevant during long-horizon simulation.

\subsection{Unbounded volatilities}

In the baseline parameterization, the individual volatilities are recovered as
\[
\sigma_i(z) = \exp\bigl([\mathcal{H}(z)]_i\bigr), \qquad i = 1,\ldots,\ell.
\]
We showed in the proof of Proposition~\ref{thm:existence_uniqueness_sde} that the bounded activation $\psi$ confines the pre-exponential outputs to a compact interval $[-C_{\mathrm{pre}}, C_{\mathrm{pre}}]$, so that $\sigma_i \in [e^{-C_{\mathrm{pre}}}, e^{C_{\mathrm{pre}}}]$ for all $z$.
This guarantees the theoretical linear-growth and Lipschitz conditions needed for existence and uniqueness of the SDE solution, but the \emph{practical} upper bound $e^{C_{\mathrm{pre}}}$ depends on the learned weight matrices $W_1,W_2$ and can be large.
When the latent state drifts outside the support of the training data --- as inevitably happens during multi-year simulation --- the network may output pre-exponential values near its saturation limits, producing volatilities that are technically finite but economically implausible (e.g.\ $\sigma_i > 5$).

To address this, we replace the $\exp$-link with a \emph{sigmoid-bounded} parameterization:
\[
\log\sigma_i(z) \;=\; \log\sigma_{\min}
  + \bigl(\log\sigma_{\max} - \log\sigma_{\min}\bigr)
    \cdot \operatorname{sigmoid}\!\bigl(h_i(z) + b_i\bigr),
\]
where $h_i(z)$ denotes the $i$-th output of the shared hidden layer, $b_i$ is a learnable offset initialised so that $\sigma_i = \sigma_{\mathrm{init}}$ when $h_i = 0$, and $0 < \sigma_{\min} < \sigma_{\max}$ are user-specified bounds.
This guarantees
\[
\sigma_i(z) \;\in\; (\sigma_{\min},\;\sigma_{\max}) \qquad \forall\, z \in \mathbb{R}^\ell,
\]
while retaining full differentiability. The sigmoid link is monotone and its derivative is bounded, so the Lipschitz property established in the proof above still holds --- in fact with a \emph{tighter} constant, since the output range is now strictly controlled.

\begin{proposition}
\label{prop:bounded_sigma}
Let $\sigma_{\min}, \sigma_{\max} > 0$ with $\sigma_{\min} < \sigma_{\max}$. The stable volatility parameterization satisfies, for all $z \in \mathbb{R}^\ell$:
\begin{enumerate}
    \item $\sigma_i(z) \in (\sigma_{\min}, \sigma_{\max})$ for every $i = 1,\ldots,\ell$.
    \item The mapping $z \mapsto \sigma_i(z)$ is Lipschitz continuous with constant
    \[
    L_{\sigma} = \frac{1}{4}\,(\log\sigma_{\max} - \log\sigma_{\min})\,\sigma_{\max}\,\|W_2^{(i)}\|\,\|W_1\|,
    \]
    where $W_2^{(i)}$ denotes the $i$-th row of the output weight matrix.
\end{enumerate}
\end{proposition}

\begin{proof}
\emph{Part 1.} Since $\operatorname{sigmoid}(x) \in (0,1)$ for all $x \in \mathbb{R}$, we have $\log\sigma_i \in (\log\sigma_{\min},\,\log\sigma_{\max})$ and therefore $\sigma_i \in (\sigma_{\min},\,\sigma_{\max})$.

\emph{Part 2.} Write $s_i(z) = h_i(z) + b_i$ and $\phi = \operatorname{sigmoid}$. Then $\sigma_i = \exp\!\bigl(\log\sigma_{\min} + \Delta\,\phi(s_i)\bigr)$ with $\Delta = \log\sigma_{\max} - \log\sigma_{\min}$. By the chain rule:
\[
\|\nabla_z \sigma_i\| = \sigma_i \cdot \Delta \cdot \phi'(s_i) \cdot \|\nabla_z h_i\|
\leq \sigma_{\max} \cdot \Delta \cdot \tfrac{1}{4} \cdot \|W_2^{(i)}\|\,\|W_1\|,
\]
where we used $\sigma_i \leq \sigma_{\max}$, $\phi'(x) = \phi(x)(1-\phi(x)) \leq \frac{1}{4}$, and the fact that $\|\nabla_z h_i\| \leq \|W_2^{(i)}\|\,\|W_1\|$ because the hidden activation $\psi$ satisfies $|\psi'| \leq \frac{1}{4} \leq 1$.
\end{proof}


\subsection{Positive-definite correlations for arbitrary dimension}

The baseline network parameterizes correlations as
\[
\rho_{ij}(z) = \tanh\!\bigl([\mathcal{H}(z)]_k\bigr),
\]
which ensures each pairwise correlation lies in $(-1,1)$, but does \emph{not} guarantee that the full $\ell \times \ell$ correlation matrix $R$ is positive definite when $\ell \geq 3$.
A set of pairwise correlations $\{\rho_{ij}\}$ that are individually valid can still fail to be jointly realisable: for example, $\rho_{12} = \rho_{13} = 0.9$ and $\rho_{23} = -0.9$ gives a correlation matrix with a negative eigenvalue.
If $R$ is not positive definite, the Cholesky factorization $R = L_R L_R^\top$ fails and the diffusion matrix $\Sigma = \operatorname{diag}(\sigma)\,L_R$ is undefined.

For $\ell = 2$, this issue does not arise: any single correlation $\rho_{12} \in (-1,1)$ gives a valid $2 \times 2$ correlation matrix.
However, since the architecture is designed to work for any $\ell$, a structurally safe parameterization is required.

We adopt the \emph{hyperspherical Cholesky} parameterization \citep{Pinheiro1996}, which maps $\ell(\ell-1)/2$ free angles $\theta_k \in (0,\pi)$ to a lower-triangular matrix $L_R$ such that $R = L_R L_R^\top$ is a valid correlation matrix by construction.

\begin{definition}[Hyperspherical Cholesky factor]
\label{def:hyperspherical_cholesky}
For $\ell \geq 2$, define the lower-triangular matrix $L_R \in \mathbb{R}^{\ell \times \ell}$ by:
\[
L_R[0,0] = 1, \qquad
L_R[i,j] = \cos\theta_{i,j+1}\prod_{k=1}^{j}\sin\theta_{i,k}, \quad j < i, \qquad
L_R[i,i] = \prod_{k=1}^{i}\sin\theta_{i,k},
\]
where the angles $\{\theta_{i,k}\}$ are indexed row by row, using $\ell(\ell-1)/2$ free parameters in total.
\end{definition}

\begin{proposition}
\label{prop:pd_correlation}
If all angles satisfy $\theta_{i,k} \in (0,\pi)$, then $R = L_R L_R^\top$ is a symmetric positive-definite matrix with unit diagonal (i.e.\ a valid correlation matrix).
\end{proposition}

\begin{proof}
\emph{Unit diagonal.} Each row of $L_R$ has squared norm:
\[
\sum_{j=0}^{i} L_R[i,j]^2 = 1,
\]
which follows from the trigonometric identity $\cos^2\theta + \sin^2\theta = 1$ applied recursively.
Therefore $R[i,i] = \sum_j L_R[i,j]^2 = 1$.

\emph{Positive definiteness.} For $\theta_{i,k} \in (0,\pi)$, we have $\sin\theta_{i,k} > 0$, so the diagonal entries $L_R[i,i] = \prod_{k=1}^{i}\sin\theta_{i,k} > 0$.
Hence $L_R$ is lower-triangular with strictly positive diagonal, which means it is non-singular.
For any $x \neq 0$:
\[
x^\top R\, x = x^\top L_R L_R^\top x = \|L_R^\top x\|^2 > 0,
\]
since $L_R^\top$ is non-singular. Therefore $R$ is positive definite.

\emph{Symmetry.} $R = L_R L_R^\top$ is symmetric by construction.
\end{proof}

In the stable network, the angles are produced from the shared hidden layer via
\[
\theta_k(z) = \pi \cdot \operatorname{sigmoid}\!\bigl(g_k(z)\bigr) \;\in\; (0,\pi),
\]
where $g_k(z)$ is the $(d+k)$-th output of the network.
The sigmoid ensures that the angles are always in the open interval $(0,\pi)$, so Proposition~\ref{prop:pd_correlation} applies for every input $z$ and the correlation matrix is guaranteed to be positive definite.

\begin{remark}[Initialization]
When the output-layer weights are initialized to zero, we have $g_k(z) = 0$ for all $z$ and $k$, so $\theta_k = \pi/2$.
Since $\cos(\pi/2) = 0$, the off-diagonal entries of $L_R$ vanish and $L_R = I$, giving $R = I$ (zero correlations).
The model therefore starts from an uncorrelated prior and learns non-trivial correlation structure during training.
\end{remark}
